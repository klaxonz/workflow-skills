# Installing Workflow Skills for OpenCode

## Direct install

```bash
npx @klaxonz/workflow-skills install opencode
```

For a user-level install:

```bash
npx @klaxonz/workflow-skills install opencode:global
```

## Verify

Use OpenCode's skill listing command to confirm these skills are available:

- `code-audit`
- `code-fix`
- `feature-flow`

Install Workflow Skills separately for each agent you use.
